# down-worldcup-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存储，可通过 GitHub Pages 直接访问。

## 项目简介

down-worldcup-journal 是一个专注于 HTML 页面存档与展示的项目。仓库内保存了多个自包含的网页文件，每个文件独立运行，不依赖后端服务。适合作为小型网页集合的托管空间。

## 目录说明

```
.
├── index.html        # 仓库主页（可自定义）
├── pages/            # 存放归档的 HTML 页面
│   ├── example1.html
│   └── example2.html
├── assets/           # 页面共用资源（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md         # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面，文件名建议使用英文或拼音。
- `assets/` 目录存放页面所需的共用资源，避免重复引用。

## 页面归档说明

- 每个 HTML 页面应为独立完整文件，包含所有必需的结构与样式。
- 页面内容需符合法律法规及 GitHub 服务条款。
- 若页面依赖外部资源（如图片、字体），建议使用相对路径或内联方式，确保离线可查看。
- 新增页面时，请在 `pages/` 目录下创建文件，并更新主页 `index.html` 的导航链接（如有）。

## 维护说明

- 本仓库由维护者不定期更新，添加或修改归档页面。
- 欢迎通过 Issue 提交页面归档建议，但需提供完整的 HTML 文件内容或链接。
- Pull Request 接受页面优化或新增归档，请确保文件格式正确且无外部依赖问题。
- 仓库不保证所有页面长期可用，维护者有权移除不合规或长期未更新的内容。

## 许可

本仓库仅用于页面归档与展示，不提供任何形式的服务承诺。页面内容版权归各自作者所有，如涉及侵权请联系维护者处理。
